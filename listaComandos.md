# Introducción a la Terminal Linux

Comandos Básicos:

Indicar la ruta actual
```bash
pwd
```
Listar los archivos y directorios de la ruta actual
```bash
ls
```
Crear un directorio
```bash
mkdir <Nombre del directorio>
```
Cambiar directorio
```bash
cd <ruta_deseada>
```
Limpiar terminal
```bash
clear
```
